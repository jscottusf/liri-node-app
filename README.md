# Liri Node Application

The liri node app allows users to to find information via the following commands:
- concert-this
- movie-this
- spotify-this-song
- do-what-it-says

## Directions

1. Open the console

2. Navigate to the liri-node-app folder

3. Input: node liri.js + {command} + {search item}
